
======
CC2541
======

Robotice driver for SensorTag a Bluetooth Low Engery (BLE GATT) device


Requirements
------------

* gatttool
* pexpect


Usage
-----

.. code-block:: bash

    python driver.py 78:A5:04:8C:25:A7

    in while cycle

    python driver.py 78:A5:04:8C:25:A7 -r
